Welcome to Ambassy

Ambassy is designed to help parkrun Regional Event Ambassadors (REAs) to allocate events. The tool allows you to upload CSV files containing the names of the Event Ambassadors you support and the events they support.

This mockup shows how Ambassy would look with a header and footer similar to Foretoken, but using the MD4 color scheme instead of parkrun brand colors.

MD4 Color Scheme

The header uses #0d2b33 (deep teal-blue) and the footer uses #30403d (dark green-grey) from the MD4 palette. Accent colors include #3d9df2 (bright blue) for links and #ffcc00 (bright yellow) for emphasis.
